## Technologies

- **Backend:** **JAVA 21**.
- **Security:** **Spring boot Security**.
- **Authentication:** **JWT**.
- **Frontend:** **React**.
- **Database:** **MySQL**.
- **ORM:** **Hibernate**.
- **Queries:** **HQL**.
